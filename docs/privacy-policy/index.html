개인정보처리방침

VibeCodingExpert는 설명 가능한 개인화 요약 서비스를 제공하기 위해 필요한 범위에서만 개인정보를 처리합니다. 이 문서는 계정 정보, 맞춤형 설정, 저장 항목, 피드백 데이터가 어떤 목적으로 사용되고 어떻게 관리되는지 설명합니다.

적용 서비스 VibeCodingExpert 앱 및 관련 웹 페이지
시행일 2026년 3월 8일
공개 URL forevernewvie.github.io/YourIntetion/privacy-policy/

1. 수집하는 정보

  • 계정 생성 및 로그인에 필요한 이메일 주소와 인증 관련 식별 정보
  • 온보딩 주제 선택, 소스 선호도, 톤/빈도, 규칙 설정 등 개인화 구성 정보
  • 저장한 요약, 피드백 제출 내용, 서비스 품질 개선에 필요한 최소 운영 로그
  • 비밀번호 재설정, 이메일 인증 등 계정 보호 절차에 필요한 상태 정보

2. 이용 목적

  • 개인화된 digest 생성, 순위 설명, 저장 기능 제공
  • 계정 인증, 세션 유지, 비밀번호 재설정, 보안 점검 수행
  • 사용자 설정과 피드백을 반영한 서비스 품질 개선
  • 장애 대응, 오남용 방지, 법적 의무 이행

3. 보관 기간

  • 계정 정보와 개인화 설정은 서비스 이용 기간 동안 보관합니다.
  • 저장한 digest, 피드백, 규칙 설정은 사용자가 삭제하거나 계정을 종료할 때까지 유지될 수 있습니다.
  • 법령 또는 분쟁 대응상 보관이 필요한 정보는 해당 기간 동안 별도로 보관할 수 있습니다.

4. 제3자 제공 및 처리 위탁

  • 서비스 제공에 필요한 범위를 넘어서 개인정보를 판매하거나 임의로 공유하지 않습니다.
  • 호스팅, 인증, 저장소, 배포 인프라 운영 과정에서 제한된 범위의 처리 위탁이 발생할 수 있습니다.
  • 법령에 따른 요구가 있는 경우 필요한 범위에서만 정보를 제공할 수 있습니다.

5. 이용자 권리

  • 설정 화면에서 개인정보 관련 제어 항목과 정책 링크를 확인할 수 있습니다.
  • 계정 삭제, 데이터 내보내기 등 권리 행사는 서비스에서 제공되는 기능 또는 지원 채널을 통해 요청할 수 있습니다.
  • 잘못된 정보가 있다면 정정 또는 삭제를 요청할 수 있습니다.

6. 보호 조치

  • 인증 토큰, 비밀번호, 민감 필드는 로그에서 직접 노출되지 않도록 최소화합니다.
  • 서비스 운영에 필요한 접근 통제, 오류 추적, 세션 관리 절차를 유지합니다.
  • 다만 인터넷 전송 및 저장 방식의 특성상 완전한 보안을 절대적으로 보장할 수는 없습니다.

7. 문의

개인정보 처리에 관한 문의는 현재 저장소 이슈 채널을 통해 접수받습니다. GitHub Issues 바로가기

안내 이 문서는 제품 기능과 저장소 상태를 기준으로 작성되었습니다. 법적 검토가 필요한 서비스 단계에 들어가면 별도 법률 자문을 거쳐 문구를 확정하는 것이 안전합니다.
